Project: Tourism GPT using RAG

This project implements a Retrieval-Augmented Generation (RAG) powered chatbot focused on tourism information. The dashboard shows the RAG index status, recent user queries, and tools to upload local documents (PDFs, Markdown) to improve retrieval quality.

Sample Integration (concept)

// Pseudocode: query flow
query = "3-day itinerary for Karachi"
docs = retrieve(query, top_k=5)   // RAG retrieval from indexed documents
context = concat(docs)
response = LLM.generate(prompt + context)
return response
